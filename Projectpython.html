About pyTOOLs

pyTOOLs : A Bundle of python tools.

WELCOME

pyTOOLs is a platform where you can find plenty of tools that are designed using python and performs real world tasks for you. The idea is to prevent third party services for every single task, so as to make daily tasks quick , easy and secure.

NOTE: Before buying any of our products make sure you have Python Environment on your localhost. If not click the button below:




1.yTv

This is basically a tool which provides you the best and the quickest way to download your favourite youtube videos into your localhost. This tool is an example to show how powerful can Python be, if we use it on the right way with right mindset. You should definetely use this tool if you are one of those who loves , who wants to download youtube videos securely without accessing any website.



Prerequisite: Make sure you have tkinter and pytube modules installed, if not then follow the steps:

Step 1: Open cmd and type 'pip install tkinter'
Step 2: Open cmd and type 'pip install pytube'

2.python-nmap PORT scanner

This is basically a PORT scanner which is designed in Python. It uses some Python modules in order to scan ports for the input ip address and provides the result as quick as possible. It asks the user to enter a specific port range that the scanner needs to look for and results accordingly.



Prerequisite: Make sure you have nmap and ipaddress modules installed, if not then follow the steps:

Step 1: Open cmd and type 'pip install nmap'
Step 2: Open cmd and type 'pip install ipaddress'

3. Encode and Decode

This is basically a tool which provides you a GUI to perform and understand basic encryption( Converting readable text to unreadble text. ) and decryption(Converting unreadable text to readble text. ) process. You can use this tool for enhancing security for your daily tasks like secret chatting. This tool requires a key to be entered while encrypting and the same key for decrypting it as well.



Prerequisite: Make sure you have tkinter and datetime modules installed, if not then follow the steps:

Step 1: Open cmd and type 'pip install tkinter'
Step 2: Open cmd and type 'pip install datetime'
